# IoT Electrical Monitoring System

## Overview
ESP8266-based monitoring system using:
- PZEM004T (Voltage, Current, Power, Energy)
- DHT11 (Temperature, Humidity)
- Blynk IoT Cloud

## Features
- Real-time monitoring
- Cloud dashboard
- Threshold alerts
- User validation (4 users, 90% success rate)

## Setup
1. Install Arduino IDE
2. Install required libraries
3. Add your Blynk template ID & auth token
4. Upload to ESP8266
